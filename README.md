## 프로젝트 소개

### 서비스 개요

- 그룹에 소속하여 시간 관리/일정을 리마인드 시켜주는 공동 알람 시스템

### 주요 기능

- 개인 알람 설정 가능
- 그룹장이 설정 가능한 공유 알람
- 가입 유저를 닉네임 기반으로 검색 후 알람 그룹에 초대 가능



### 프로젝트의 특징

1. 그룹으로 알람 설정 기능
2. 일어나지 않은 사람에게 알람 보내기 기능
3. 서버에서 알람 정보가 관리되기 때문에 자유롭게 기기 간 알람 이동 가능
4. 중복 로그인시 자동 로그아웃 기능
